# DPM-KM Udinus – Website Resmi

Website resmi Dewan Perwakilan Mahasiswa – Keluarga Mahasiswa Universitas Dian Nuswantoro periode 2025/2026.

## Struktur Folder
- `index.html` – Halaman beranda
- `assets/` – Aset statis (CSS, JS, gambar, dokumen, icons)
- `pages/` – Seluruh halaman internal

## Teknologi
- HTML5 + CSS3 (custom)
- JavaScript (vanilla)
- Google Apps Script (untuk backend aspirasi)

## Cara Menjalankan
Cukup buka `index.html` di browser atau deploy ke GitHub Pages.
